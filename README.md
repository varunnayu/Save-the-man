# 🎯 Save The Man - Hangman Game

🔗 **Live Demo:** https://varunnayu.github.io/Save-the-man/

## 📌 Overview

**Save The Man** is a classic Hangman-style word guessing game built with a clean and interactive UI.
The player must guess the correct word before the man is fully drawn — every wrong guess brings him closer to danger!


## 🚀 Features

* 🎮 Interactive gameplay
* ✍️ Letter-by-letter guessing system
* ❌ Wrong guess tracking with visual updates
* 🎯 Win / Lose detection
* 🔄 Restart game functionality
* 📱 Fully responsive design


## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)


## 🎨 UI Highlights

* Clean and minimal design
* Smooth user interaction
* Visual feedback for guesses
* Mobile-friendly layout

1. Clone the repository


2. Open the folder
3. Run `index.html` in your browser
